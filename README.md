## 🎯 Overview

Interview IQ is an AI-powered career preparation platform that helps job seekers understand their readiness for a target role before they apply or interview.

By analyzing a candidate's **Resume**, **Job Description**, and **Professional Summary**, Interview IQ generates personalized reports, evaluates job fit, identifies skill gaps, and provides actionable recommendations to improve interview success.

Whether you're preparing for your first job or your next big career move, Interview IQ helps you approach interviews with confidence and clarity.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/akashdatkar/interview-iq.git
cd interview-iq
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
DATABASE_URL=
AI_API_KEY=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
```

### Run Development Server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```
